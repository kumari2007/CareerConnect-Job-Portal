# CareerConnect – Job Portal Website

## 📌 Project Overview

CareerConnect is a full-stack Job Portal web application developed using PHP, MySQL, HTML, CSS, JavaScript, and XAMPP.
The platform allows users to search and apply for jobs, while admins can manage job postings and view applicants.

This project was developed as part of the ApexPlanet Software Internship Task 5.
